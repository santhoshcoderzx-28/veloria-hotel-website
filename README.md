# Veloria Grand Hotel — MANTRA 2026 Assignment 1

A fully responsive, SEO-optimised static hotel website built with pure HTML5, CSS3, and JavaScript.

## Live Demo
🌐 [View on Netlify](https://veloria-hotel.netlify.app/)

## Pages
| Page | File | Purpose |
|------|------|---------|
| Home | `index.html` | Hero, booking bar, rooms preview, amenities, testimonials |
| About | `about.html` | Hotel story, values, team, awards, FAQ accordion |
| Rooms & Packages | `rooms.html` | All room categories and curated packages |
| Gallery | `gallery.html` | Photo gallery with lightbox feature |
| Contact / Booking | `contact.html` | Booking form with JS validation, map, contact info |

## Tech Stack
- **HTML5** — Semantic structure with proper heading hierarchy
- **CSS3** — Flexbox/Grid layout, CSS variables, media queries for responsive design
- **JavaScript** — Mobile menu toggle, scroll-reveal animations, form validation, lightbox, FAQ accordion, booking bar

## Features Implemented
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Fixed navbar with scroll effect
- ✅ Mobile hamburger menu
- ✅ Quick booking bar on homepage
- ✅ Scroll-reveal animations
- ✅ Image lightbox on gallery page
- ✅ FAQ toggle accordion
- ✅ Full form validation with error messages
- ✅ Internal linking across all pages
- ✅ SEO: `<title>`, meta description, meta keywords, proper heading structure (`h1` → `h2` → `h3`), alt text on all images
- ✅ Accessibility: ARIA labels, roles, semantic HTML

## How to Deploy on Netlify
1. Push this repository to GitHub (make it **Public**)
2. Go to [netlify.com](https://netlify.com) → Log in
3. Click **"Add new site"** → **"Import an existing project"**
4. Connect your GitHub account and select this repository
5. Click **"Deploy site"**
6. Your live URL will appear in seconds ✅
